不和 (fuwa)

不和 (fuwa, meaning "discord" or "disagreement") is a small Discord API wrapper made as a kind of experiment and learning experience. Originally written in JavaScript, it is now being re-written in TypeScript.

Note: This library is not published to the npm registry, yet. It is planned to publish it once basic functionality is implemented.

Installation

Node.js version 14.x.x or higher is required for this library.

Installing with pnpm, npm or Yarn is simple. I don't know any other package managers.

# with npm
npm install --save nearlysplat/fuwa

# with yarn
yarn add github:nearlysplat/fuwa

# with pnpm
pnpm add nearlysplat/fuwa

Usage

While developing Fuwa, I was aiming for a near-discord.js or Eris feel.

An example client is this:

import { Client, Intents } from "fuwa";
// replace this with your token
const client = new Client("my_token", {
  intents: Intents.NON_PRIVILEDGED // non priviledged intents
});

client.on("ready", () => {
  console.log(`Logged in as ${client.user.username}#${client.user.tag}!`);
});

client.on("messageCreate", (msg) => {
  if (msg.content === "!ping") msg.channel.messages.create("Pong!");
})

client.connect();
